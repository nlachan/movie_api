Movie API

Server-side component of a “movie” web application.
The online platform will grant users access to details about various films, directors, and genres.
Users can register, modify their personal information, and compile a list of their preferred movies.


Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get list of movies /movies GET None An array of JSON objects holding data about all the movies
Get list of movies by director /movies/directors/[director name] GET None An array of JSON objects holding data about all the movies by requested director
Get list of movies by year of release /movies/releaseYear/[year] GET None An array of JSON objects holding data about all the movies by requested year of release
Get list of movies by genre /movies/genres/[genre] GET None An array of JSON objects holding data about all the movies by requested genre
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie like this:
{ 'id': '######',
'title': 'Iron Man',
'director': 'John Favreau',
'release-year': '2008',
'genre': 'action' }
Add a movie /movies POST A JSON object holding data about the movie to add, structured like:
{ 'title': 'Iron Man',
'director': 'John Favreau',
'release-year': '2008',
'genre': 'action' } 		A JSON object holding data about the user that was added, including an ID:
{ 'id': '######',
'title': 'Iron Man',

'release-year': '2008',

Remove a movie by ID /movies/id/[id] DELETE None A text message indicating whether the movie was successfully removed
